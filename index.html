2. Multi-Robot Relay Race Simulation (ROS2 & CoppeliaSim)

Python ROS2 (Humble) CoppeliaSim Pure Pursuit TF2 YAML

Project Overview

Developed and simulated a cooperative relay race between two autonomous mobile robots using the Turtlebot3 model. The system implements a coordination architecture where one robot finishes a trajectory and triggers the second robot to continue autonomously.

Key Technical Contributions
  • Path Tracking Control: Implemented Pure Pursuit trajectory control using cmd_vel and odom.
  • Multi-Robot Architecture: Designed isolated ROS2 namespaces (/robot1 and /robot2) to avoid topic collisions.
  • Coordination Logic: Integrated a custom relay_manager node for synchronization between robots.
  • Waypoint System: Created a dynamic waypoint publisher using YAML configurations.
